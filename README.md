## 🚀 Tentang Project

ERP Toko System adalah aplikasi berbasis web yang dibangun menggunakan Laravel untuk membantu pengelolaan toko, meliputi:

- Manajemen penjualan
- Manajemen pembelian
- Manajemen stok barang
- Laporan keuangan & bisnis
- Dashboard real-time

---

## 🔧 Teknologi

- Laravel 11
- PHP 8+
- MySQL
- Tailwind CSS (jika pakai)
- Vite
